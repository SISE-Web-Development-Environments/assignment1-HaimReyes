Q&A

Three main differences between web1, web2 and web3
first:

web1 is client-server based while web2 and web3 can be p2p based

second:

web1 and web2 are are design for millions to billions of users while web3 is designed for trillions of users

third:

web1 and web2 are design for the human user while web3 is also designed for machins

Two example use cases
first:

Downloading a file:
In web1 the the file data will transfer from the host to the client (client-server).
In web3 and 3 the file data can be transfered from multiple sources (p2p).

second:

consuming news:
In web1 the user can only read pre-loaded articles (meaning there is a need to refresh the page in case something new happened while browsing).
In web3 the user can read "breaking news" articles (freshly uploaded articles) while the page is already loaded (and even receive a notification).
